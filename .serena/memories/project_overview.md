# U Stacks Project Overview

U Stacks is a collection of modern full-stack starter templates for web applications. It provides three different stack options:

## Available Stacks

### 1. Tails Stack
- **Framework**: Next.js 15 (App Router) with Turbopack
- **UI Components**: Radix UI with shadcn/ui
- **Styling**: Tailwind CSS v4
- **Database**: Prisma ORM
- **Authentication**: Better Auth
- **Deployment**: Cloudflare Pages via OpenNext
- **Formatting/Linting**: Biome

### 2. Sonic Stack
- **Framework**: React Router v7 (Meta-framework)
- **Backend**: Hono server with hono-react-router-adapter
- **UI Components**: Mantine
- **Styling**: Tailwind CSS with Mantine presets
- **Database**: Prisma ORM
- **Authentication**: Clerk
- **Deployment**: Fly.io
- **Formatting/Linting**: Biome

### 3. Hono on React Router
- **Framework**: React Router v7 with Hono backend
- **Backend**: Hono with hono-react-router-adapter
- **Styling**: Tailwind CSS
- **Database**: Prisma ORM
- **Deployment**: React Router Serve
- **Formatting/Linting**: Biome

## Common Technologies
- TypeScript
- Biome for code formatting and linting
- Prisma for database management
- Zod for validation
- React 19
