# U Stacks

## How to use

### Shadow Stack

```shell
npx degit yicru/u-stacks/shadow my-app
cd my-app
bun scripts/setup.ts
```

### Tails Stack

```shell
npx degit yicru/u-stacks/tails my-app
```

### Sonic Stack

```shell
npx degit yicru/u-stacks/sonic my-app
```

### Hono on React Router

```shell
npx degit yicru/u-stacks/hono-on-react-router my-app
```
